### How to add css file to html code

* for example 

``` html

<html>
  <head>
    <!--- import link about css directory --->
    <link rel="stylesheet" href="css/style.css">

  </head>
</html>


``` 

# components of css 

* css에는 여러가지 컴포넌트들이 있다. 
* 크게 세 가지로 구분되는데, 
  1. 선택자
  2. 속성명
  3. 속성값
* example

``` Css
h1 {
color: red;
}
```
> 이때,
> > h1 ->선택자
> > color -> 속성명
> > red -> 속성값
